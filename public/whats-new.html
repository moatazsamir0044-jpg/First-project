SEO & Marketing — Exactly How It's Applied

The current site relies entirely on paid traffic or word-of-mouth. The new version builds a compounding organic channel — every listing page is a landing page that Google can index, rank, and show with rich visuals. Here is exactly what was implemented and why it matters.
Google

Schema.org Structured Data on Every Listing Page

Each property page emits a JSON-LD block that Google reads to understand the content. This is what makes Google show star ratings, price, and availability directly in search results — without the user having to click.

  • LodgingBusiness — property name, address, coordinates, number of rooms, all amenities
  • AggregateRating — star score + review count shown as yellow stars in Google
  • Review — up to 5 individual guest reviews indexed by Google
  • Offer — price per night in EGP, currency code, availability status
  • BreadcrumbList — Home → Listings → Area → Property Name shown in the URL path below the result

Impact: Listings with rich snippets earn 20–30% higher click-through rates from Google — at zero ad spend.

Google

Schema.org on the Homepage — Sitelinks Searchbox

The homepage emits an Organization, WebSite, and SearchAction schema. The SearchAction tells Google about the site's search capability.

  • Sitelinks searchbox — when someone Googles "BirdNest," a search box can appear directly in the result, letting users search property listings without visiting the site first
  • Organization identity — logo, name, area served (Egypt), languages (English + Arabic), contact details — establishes BirdNest as a known entity in Google's Knowledge Graph
  • RealEstateListing type with all four cities listed as areaServed
Social / WhatsApp

Auto-Generated Open Graph Preview Images

Every page has an Open Graph image generated in code using the BirdNest brand fonts (Recoleta + Gilroy) and colors. When any link is shared on WhatsApp, Facebook, LinkedIn, or iMessage, it shows a professional branded card — not a blank preview.

  • Homepage OG image — "Your Home Away From Home in Egypt," location chips (New Cairo · North Coast · El Gouna · Sheikh Zayed), BirdNest logo and URL
  • Per-listing OG image — property photo at 1200×630px, listing title, location, BirdNest branding
  • All pages — title tag, meta description, canonical URL, og:url, og:type set dynamically

Why this matters: WhatsApp is the primary sharing channel for Egyptian property seekers. A rich preview card vs a bare URL increases click-through dramatically — and every share is free marketing.

Page Speed

Core Web Vitals — Google Ranking Factor

Google officially uses page speed as a ranking signal. Three specific improvements were made:

  • Next.js Image component — all property photos auto-converted to WebP (30–50% smaller), lazy-loaded, correct dimensions served per device. Eliminates the "image too large" Lighthouse warning.
  • Font subsetting — Recoleta and Gilroy are subset to only the characters used and served from the CDN edge. No layout shift (CLS = 0).
  • Static generation — listing pages are pre-rendered at build time. First byte is served from the edge in milliseconds, not from a database query.

Impact: Faster pages rank higher and convert better — especially on mobile connections in Egypt where network speeds vary.

Bilingual SEO

Arabic & English as Separate Indexed Pages

Arabic content lives at /ar/... routes (e.g., /ar/listings, /ar/how-it-works). These are full separate pages — not just CSS translations — which means:

  • Google indexes both the English and Arabic versions as separate, properly-translated pages
  • Arabic search queries ("شقق مفروشة القاهرة الجديدة") can rank and show the Arabic page directly
  • RTL layout is correct — Arabic readers don't get a mirrored-looking page
  • All Arabic pages have Arabic meta titles and descriptions set — not auto-translated

Impact: Captures Arabic-language organic search traffic — a segment most competitors ignore entirely.

Marketing

Newsletter Capture & Email Channel

An email signup section on the homepage captures leads directly — no third-party pop-up tool needed. Combined with the user account system:

  • Registered users' emails are in the database — North Coast inventory can be promoted to them before peak season
  • Wishlist data identifies high-intent guests — email them when a saved property has availability
  • Cost per booking from owned email lists is a fraction of any paid channel
